# AI Resume Analyzer

AI Resume Analyzer is a Flask-based web application that compares a resume with a job description using Natural Language Processing (NLP). It calculates a resume match score and identifies matching and missing skills through an intuitive, modern user interface.

## Features

- Upload PDF resumes
- Compare resumes with job descriptions
- Resume match score
- Matching & missing skills analysis
- Responsive multi-page interface

## Tech Stack

- Python
- Flask
- Scikit-learn
- PyPDF2
- HTML
- CSS
- JavaScript
